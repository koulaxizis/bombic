Bonuses

Like every other bomberman game also our Bombic has many bonuses, which helps you to kill your enemies easyer. We tryed to add all important and known. Bonuses will be avalable in sigle player in this ored. Some of them can be used only in deathmatch because they are too strong. Bonus apears on destroyed ground. In storyplaing the apears randomly and in deathmatch allways. They can be devided in still bonuses (it is enough to take them once) time bonuses (their effect disapear after a little time) and one time bonuses. You can have only one time bonus at one time. If you have time bonus and you take another time bonus, the effect of the onld one will disapear and only the new one is active. It is very usefull when you have illness - it can be cured this way.

Basic set

Dead
Anti-bonus. You will be ill for a little time. You can be slowed down or unable to put bombs or you can go berserk (still bombing) or many other illnesses. (You will see)
Bomb
It raises number of bombs that you can use. Maximal number of bombs is 9.
Fire
It raises range of your bombs explosion. Maximal fire range is 9.
Speer
It raises speed of your bombic. It can be picked up 4X.
Distant explode
It is time limited bonus. You press button for putting bomb. You put the bomb and while you are holding the button, the bonb won't explode. When you release the button, you bomb will explode. It is extremly strong with sending ir kicking. But it is easy to do a suicide.
Megabomb
After picking up this bonus you will receive two megabombs. They have huge explosion.
Kicking
It is still bonus. When you try to run on place, where is bomb, the bomb will start moving. It is possible to kick into bomb tah is allready moveing.
Napalm
After picking up this bonus you will receive two napalmbombs. It explodes like normal bomb, but the ground is burning on the place of explosion.
Sending
With this still bonus you will be able to send your bombs. If you are stangind still (not moveing) and put a bomb the bomb will start to move the direction you are looking. When you are moveing, the bonus has no effect.
Shield
Time limited bonus. It makes you immortal. If you are stanging on place where mortal would die, you can't put bombs.


Deathmatch only

Live
Gives you one more live. Number of lives is unlimited.
Green Dead
The most insidius bonus in game. When you pick it, all other players will receive Deadbonus.
FireMan
Time limited bonus. It turns your bombic into killing machine. When you press button for putting bomb, the fire will apear around you. You are immortal with this bonus.
 



 

Monsters

You need some evil monster to play on your own. We have prepared a lot of them for you. The differ in speed, intelligence, number of lives and by picture. If the monster has more lives, it lessens after every hit.

Grey ball
The weakest and the easyest enemy. It is easy to get rid of it.
Slimer
It is slower than red ball. but it has two lives and it doesn't step on places where is fire. It isn't problem to kill it.
Clouder
Tought competitor. Isn't totaly slow and it is inteligent. It isn't easy to kill it. Fortunately it has only one live. Because it is flying it can move over abyss.
Trap
A rolling ball. It changes direction only after collision. It has two lives. It needs to be patient when you are trying to destroy it. If you hurry, it will kill you.
Red ball
Family related to the Grey ball, but it has two lives and little bit more intelligence. To kill it isn't too difficult.
Advenced trap
It is more aggresive than classic trap. It is fast and has advenced intelligence. It is trying to kill you when you are too near.
Sandmaster
It has three lives and it isn't stupid, but it is sloooow. It is a good idea to be carefull.
Mooner
Very aggressive and fast enemy. But it has only one live. It is the most dangerous enemy. It is a hard work to kill it.
Ghost
First bad hero. It is like Clouder, but it has more lives.
TV
Second bad hero. It is clever, fast, it has three lives and it shoots lightnings.
The evilest bad guy
We want you to discover The evilest bad guy on your own.
 


Editor

  We decided to give you with the game also editor. You can completly change the game using this editor. But it can be dangerous to use it. We recomend you to backup game data before you demage them. You use editor at your own risk. Maps can be saved in directory maps or in the same directory as bombic.exe. Originaly the editor was only for us, so it isn't too user friendly. When you create deathmatch map, don't forget to set the map as deathmatch in menu->file->properties.
